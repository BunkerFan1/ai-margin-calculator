# Launch Checklist for AI Margin Calculator

## 🎯 Pre-Launch (1 Week Before)

### Technical Setup
- [ ] **Domain Setup**
  - [ ] Register domain (suggestions: ai-margin-calculator.com, aiuniteconomics.com, marginai.tools)
  - [ ] Configure DNS
  - [ ] Set up SSL certificate

- [ ] **Hosting Deployment**
  - [ ] Deploy to Vercel/Netlify
  - [ ] Test on production URL
  - [ ] Configure custom domain
  - [ ] Enable CDN/caching

- [ ] **Performance Testing**
  - [ ] Run Lighthouse audit (target: 95+ score)
  - [ ] Test on slow 3G connection
  - [ ] Verify mobile responsiveness
  - [ ] Cross-browser testing (Chrome, Safari, Firefox, Edge)

### Content Preparation
- [ ] **Social Media Assets**
  - [ ] Create preview image (1200x630px for og:image)
  - [ ] Design Product Hunt gallery images (5-8 screenshots)
  - [ ] Record 30-second demo video/GIF
  - [ ] Write Twitter thread (5-7 tweets)

- [ ] **Launch Copy**
  - [ ] Product Hunt title (60 chars max)
  - [ ] Product Hunt tagline (60 chars max)
  - [ ] Product Hunt description (250 words)
  - [ ] HN Show title: "Show HN: [Title]"
  - [ ] LinkedIn post
  - [ ] Email announcement

### Analytics & Monitoring
- [ ] **Add Analytics** (choose one)
  - [ ] Google Analytics GA4
  - [ ] Plausible Analytics
  - [ ] Simple Analytics
  - [ ] Fathom Analytics

- [ ] **Error Monitoring** (optional)
  - [ ] Sentry.io integration
  - [ ] LogRocket for session replay

- [ ] **Feedback Collection**
  - [ ] Add feedback widget (Tally, Typeform, or custom)
  - [ ] Set up feedback email address
  - [ ] Create GitHub issues template
